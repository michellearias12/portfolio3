Executive Overview

This project focused on using machine learning to study risk factors associated with gender-based violence. The challenge was to move from a very large, complex survey dataset to a clean, defensible modeling workflow capable of producing insights that are both statistically grounded and socially meaningful.

Data Preparation

The first stage of the project was building a usable analytical base from a high-dimensional survey. That required reducing noise, handling missingness carefully, and converting complex raw inputs into a structured dataset suitable for machine learning.

What I Did

  • Worked with a large national survey dataset containing 9,500+ individuals and 1,000+ variables.
  • Cleaned and structured the data, including handling missing values and simplifying variable definitions.
  • Reduced the original survey into a smaller, modeling-ready dataset while preserving analytical value.
  • Prepared the data so that later feature selection and predictive modeling steps were reliable and interpretable.

Core Challenge

Large social datasets often contain overlapping questions, categorical complexity, sparse values, and variables that are difficult to operationalize. The project required balancing statistical rigor with practical usability, especially because the goal was not just prediction, but meaningful interpretation in a sensitive social context.

High-Dimensional Data Missing Values Variable Simplification Interpretability

From Survey Complexity to Actionable Signal

The strength of the project was not only in building models, but in designing a full workflow that could convert a massive and complex survey into a smaller, analytically coherent system for risk estimation and insight generation.

Feature Selection & Modeling

After preparing the data, the project focused on selecting the most relevant variables and testing predictive models that could estimate the likelihood of gender-based violence while remaining methodologically sound.

01

Feature Reduction

Applied statistical methods to identify the most relevant variables and reduce dimensionality without losing key information.

02

Model Development

Built predictive models to estimate risk and compare different machine learning approaches.

03

Model Comparison

Tested multiple alternatives and selected the best-performing models based on standard evaluation criteria.

04

Validation

Evaluated results using established validation techniques to ensure the findings were credible and robust.

Insights Generated

  • Identified key factors linked to GBV risk, including relationship history, prior violence, and socio-economic conditions.
  • Translated model outputs into clear, policy-relevant findings rather than only technical performance metrics.
  • Showed how machine learning can support understanding of sensitive social issues when paired with careful interpretation.

The real value of this work was making advanced modeling useful for public understanding — turning complex data into insights that can inform prevention, not just prediction.
